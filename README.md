# План питания — «Моя лошадь»

Исследовательский проект и веб-калькулятор для ориентировочного расчёта плана питания лошади.

## О проекте

Этот проект посвящён правильному и сбалансированному кормлению лошадей.  
Он был создан на основе исследовательской работы о рационе лошади, сухом веществе и особенностях кормления животных разных категорий: жеребцов-производителей, кобыл, рабочих лошадей и молодняка.

Практическим результатом проекта стал веб-калькулятор, который помогает рассчитать:

- массу лошади по промерам;
- суточную норму сухого вещества;
- ориентировочное количество сена и овса в сутки.

## Как пользоваться калькулятором

1. Откройте сайт проекта по ссылке выше.
2. Выберите категорию лошади:
   - жеребец-производитель;
   - кобыла;
   - рабочая лошадь;
   - молодняк.
3. Укажите нужные параметры:
   - либо готовую массу лошади,
   - либо промеры для расчёта массы.
4. Если используется расчёт по промерам:
   - **L** — длина лошади;
   - **V** — обхват груди.
5. Выберите состояние или тип нагрузки, если это требуется.
6. Нажмите кнопку расчёта.
7. Калькулятор покажет:
   - массу лошади;
   - норму сухого вещества;
   - ориентировочное количество сена;
   - ориентировочное количество овса.

## Как устроен расчёт

В основе калькулятора лежат следующие принципы:

- расчёт массы лошади по промерам;
- расчёт суточной потребности в сухом веществе;
- схема рациона:
  - 80 % фуража;
  - 20 % концентратов;
- в качестве примера используются:
  - сено — 80 % сухого вещества;
  - овёс — 86 % сухого вещества.

## Для чего создан проект

Проект создан для того, чтобы сделать тему кормления лошадей более понятной и наглядной.  
Он может быть полезен:

- начинающим конникам;
- владельцам лошадей;
- школьникам и студентам, интересующимся биологией, зоологией и коневодством;
- всем, кто хочет лучше понимать основы составления рациона.

## Важно

Калькулятор даёт **ориентировочный** результат и не заменяет консультацию:

- ветеринарного врача;
- зоотехника;
- специалиста по кормлению лошадей.
